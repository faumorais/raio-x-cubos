Como adicionar um novo censo

Este arquivo foi feito para ser atualizado sem precisar saber código. Basta editar o bloco de dados no topo do arquivo HTML.

Passo 1 — Abrir o arquivo

Abra o arquivo raio_x_di_cubos_v2.html em qualquer editor de texto (Notepad, TextEdit, VS Code, etc).

Passo 2 — Encontrar o bloco comentado

Procure o trecho que começa com:

// ─── PRÓXIMO CENSO — copie o bloco acima e preencha aqui ───
// {
//   id: "c251",

Remova as barras // do início de cada linha (isso "ativa" o bloco) e preencha os números.

Passo 3 — Preencher os dados

  • ativos: total da lista de funcionários menos sócios e desligamentos pendentes
  • respondentes: quantas pessoas responderam o formulário
  • identidade: número absoluto (não percentual) de cada grupo entre os respondentes
  • negras: pardas + pretas, seguindo classificação IBGE
  • lideranca: percentuais calculados sobre os líderes respondentes
  • acolhimento: médias em escala 1–5 e percentuais de bem-estar
  • null: use quando o dado não existir naquele censo

Passo 4 — Adicionar insights

No campo insights, escreva cada observação como texto livre. Use o tipo:

  • "positivo" — avanços e conquistas
  • "alerta" — pontos críticos que precisam de ação
  • "atencao" — pontos de atenção moderados

Passo 5 — Salvar e abrir

Salve o arquivo e abra no navegador. O novo censo aparece automaticamente como uma nova aba no painel.

Para compartilhar via Notion

Hospede o arquivo gratuitamente no GitHub Pages ou Vercel e cole o link no Notion usando /embed. O painel fica acessível para toda a equipe pelo navegador.
